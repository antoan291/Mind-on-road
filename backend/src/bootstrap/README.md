# Bootstrap

Тук живее startup composition слой на приложението.

## Какво влиза тук

- HTTP bootstrap extensions
- GraphQL bootstrap configuration
- Swagger/OpenAPI bootstrap setup
- global pipes, filters и interceptors wiring
- startup-only composition logic, която не е домейн логика

## Какво не влиза тук

- бизнес правила
- Prisma repository код
- feature-specific services
