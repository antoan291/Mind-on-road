# DevOps And AWS Knowledge

## Purpose

This folder is a knowledge hub for senior deployment, DevOps, and AWS work.

Use it for tasks related to:

- deployment architecture;
- AWS resource design;
- Docker and runtime packaging;
- CI/CD;
- monitoring, logging, and alerting;
- backup, restore, and disaster recovery;
- infra cost/risk trade-off decisions.

## Rule

Do not copy AWS architecture blindly from generic books. Always filter it through the real deployment context of this project, security requirements, backup/restore model, operational simplicity, and budget/cost constraints.
