# Frontend документация

## Цел

Тази папка съдържа frontend стандартите и реда за имплементация за проекта.

Тя не е основният продуктов source of truth.  
Целта й е frontend имплементацията да остане:

- консистентна;
- технически силна;
- лесна за разбиране;
- подравнена с архитектурата на проекта.
